## ③開発の具体的な進め方

### 現状のプロトタイプ

bublysは既に動作するプロトタイプが存在する。以下が実装済みである。

- **bublys OS本体**: バブルの生成・レイヤー管理・ルーティング・ドラッグ&ドロップ基盤
- **CSVインポーターバブリ**: CSVファイルのインポート・スプレッドシート編集・Google Sheets連携・世界線統合（操作ごとの自動コミット・undo/redo）
- **オブジェクトトランスフォーマーバブリ**: PlaneObjectからドメインオブジェクトへの自動マッピング・変換エンジン
- **囲碁バブリ**: 世界線分岐の応用例（対局の分岐・感想戦）
- **世界線グラフ基盤**: CAS + DAGによる汎用状態バージョニング、Redux + IndexedDB二層永続化

### 克服すべき課題と解決策

| 課題                 | 解決策                                      |
| ------------------ | ---------------------------------------- |
| 世界線データの肥大化         | CASによる重複排除、IndexedDBでの階層的GC              |
| バブリ間の型安全なデータ連携     | ドメインレジストリによる実行時型解決 + PlaneObjectによる疎結合連携 |
| 一般ユーザーへの世界線概念の理解促進 | DAGの視覚的表現の工夫、自動コミットによる透過的バージョニング         |
